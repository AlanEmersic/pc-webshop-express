# PC Webshop

## Frontend - Angular
Install all packages using command:
`npm i`
or
`npm install`

Run using command:
`ng s -o`
or
`npm start`

- url: **http://localhost:4200**

## Backend - Node.js Express

MySQL database

- run **schema.sql** script

- run **data.sql** script

- install all packages: 
   `npm i`
    or
    `npm install` 

- run application:
  `npm start`
